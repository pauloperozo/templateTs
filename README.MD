## Proyecto "Hfc-express-api" :rocket:

## Description

Este es un template de prueba que utiliza Node.js, TypeScript, TypeORM y PostgreSQL entre otros...

## Installation

```bash
$ npm install
```

## Running

```bash
# development
$ npm run dev

# production mode
$ npm run start

# Build
$ npm run build
```

## Test

```bash
# unit tests
$ npm run test
```
